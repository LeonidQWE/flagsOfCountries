# Flags of countries project

Project on drawing flags of countries (all countries, one country, search by name and region).

API: https://restcountries.com/#api-endpoints-v2-all

Install project:

`
npm init
`

Start project:

`
npm run dev
`

Technologies:

- React
- Typescript
- React Router
- Redux (RTK)
- Axios
- SASS
